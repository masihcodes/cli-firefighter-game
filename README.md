## 🎮 Overview

**Firefighter Rescue Simulator** is a terminal-based game where players rescue victims from grid-based maps while navigating around obstacles and walls.

Built with **Software Engineering best practices**, this project demonstrates clean Object-Oriented Programming (OOP), dynamic JSON data handling, and containerized cloud deployment. Users can run the game locally in their terminal or play it live in a web browser!

---

## ✨ Key Features

- **🧩 Object-Oriented Architecture:** Clean separation of concerns using modular Python classes and methods.
- **🗺️ Data-Driven Level Design:** Game maps, player coordinates, and obstacles are loaded dynamically from standard `JSON` files—no hardcoded game logic.
- **🎲 Procedural Map Generator:** Includes a custom script (`generate_maps.py`) that automatically builds 10 unique, solvable grid levels with dynamic borders and walls.
- **💾 State Management:** Save and load your game progress reliably using JSON serialization with full error handling.
- **☁️ Live Browser Terminal:** Containerized with **Docker** and `ttyd`, allowing anyone to play the CLI game directly inside a web browser without local installation.
